## Specialist Flags (gotchas git diff won't surface)
- **`SubprocessSandbox` is an INTERIM.** It runs self-authored code in a timed host subprocess with NO isolation. The **WSL2-isolated runner (no-network + egress allowlist + resource limits) is required before any *externally*-authored capability is trusted** — swaps in behind the `SandboxRunner` protocol. Security gate, not optional.
- **`CapabilityStore.retrieve` is lexical**, not semantic — interim until the memory slice brings an embedder; swaps in behind the same signature.
- The **schema-normalization shim (v2-01) permanently fixes** the strict-vs-lenient structured-output break that started the pivot; the **Windows `.cmd` codex-launch fix (`shutil.which`)** is baked into `codex_provider.py`.
