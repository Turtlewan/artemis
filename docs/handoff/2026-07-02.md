## Specialist Flags
- **SECURITY/CORRECTNESS — the big one.** CLI-routed models (`claude sonnet`/`opus` via `claude -p`) wrap structured JSON output in a ` ```json ` markdown fence, so every `json.loads`/`model_validate_json` call routed through the claude provider was silently failing — degrading the web-tool reader+synth, spine planning, memory consolidation, capability forge, AND the eval judge on its claude failover path. Masked in normal use because the router is codex-primary (codex uses native `--output-schema`, no fence). Fixed in `8023bbd` (strips a whole-output fence, schema calls only). The eval is what caught it.
- **Fetcher limitation:** the no-JS pinned-IP `TrafilaturaFetcher` cannot extract Wikipedia/Britannica/PubChem (bot-challenge/JS-stub → empty text). The corpus worked around it with other quality sources (NOAA/NASA/NIH/gov/edu). Affects any future real-capture work.
