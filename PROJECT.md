# Artemis

## Vision
A personal assistant that integrates with everything — a "Jarvis-like" assistant in spirit —
with a RAG-heavy **second brain** as its knowledge subsystem. A multi-subsystem platform:
assistant / orchestration + integrations + knowledge core + interface(s).

## Users
Primary: the owner (a personal assistant first). Broader users TBD — pending SP0 requirements.
